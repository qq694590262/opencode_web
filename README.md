# OpenCode Web - 企业管理系统

一个基于 Vue 3 + Spring Boot 的全栈企业管理系统。

## 技术栈

### 前端
- Vue 3 + Vite
- Vue Router
- 原生 CSS（浅蓝色主题）
- Emoji 图标

### 后端
- Spring Boot 3.1.6
- MyBatis-Plus 3.5.5
- MySQL

## 快速开始

### 前端
```bash
cd frontend
npm install
npm run dev
```

### 后端
```bash
cd backend
mvn spring-boot:run
```
